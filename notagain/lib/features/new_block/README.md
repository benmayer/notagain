///

/// - Create new app/website blocking rules
/// - Set schedules (time windows, specific days)
/// - Configure break intervals
/// - Enable/disable rules
/// - View active blocking rules
/// - Delete or modify existing rules
/// 
/// Files in this feature:
/// - screens/: Rule creation/editing screen, rules list screen
/// - widgets/: Rule form components, schedule picker, rule cards
/// - providers/: Rule CRUD operations, active rule tracking
/// - models/: BlockingRule model with scheduling info
/// - services/: Communication with blocking service for enforcement
